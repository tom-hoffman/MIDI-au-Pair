Arduino Pin Usage
-----------------

### SparkFun MIDI Shield


Hardware Serial -> MIDI IN/OUT
* LED's -> D6, D7
* Buttons -> D2, D3, D4
* Pots -> A0, A1

### Adafruit Alphanumeric Backpack

* Clock -> A5
* Data -> A4

### Indicator LED's

* Left -> D11
* Right -> D12
